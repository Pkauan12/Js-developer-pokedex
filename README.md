# 🎮 Pokédex

A responsive Pokédex application built with **HTML**, **CSS**, and **JavaScript**, consuming data from the **PokéAPI** through HTTP requests using the Fetch API.

This project was developed to practice front-end fundamentals, API integration, asynchronous JavaScript, and dynamic rendering of data in the browser.

---

# 🚀 Features

✅ Dynamic Pokémon rendering
✅ Responsive layout with CSS Grid
✅ Pokémon type-based colors
✅ API consumption with Fetch API
✅ Pagination with "Load More" button
✅ Dynamic DOM manipulation
✅ Organized and scalable structure

# ⚡ How It Works

The application sends HTTP requests to the PokéAPI, receives Pokémon data in JSON format, and dynamically renders the cards on the screen.

```txt
Browser
   ↓
HTTP Request (Fetch API)
   ↓
PokéAPI
   ↓
JSON Response
   ↓
JavaScript Processing
   ↓
Dynamic HTML Rendering
```
